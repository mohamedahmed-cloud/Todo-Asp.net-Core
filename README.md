# Todo
### Features
- Login & sign in
- Authentication
- Authorization
- Initial page for Users who haven't logged in yet.
- Logout
- Todo page has all CRUD Operation
- Category Page has all CRUD Operation
- UserINSystem page for only Admin `can Update User Role & Delete User`
